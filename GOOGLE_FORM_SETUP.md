# Google Form Integration Setup

## Steps to Set Up Google Form:

1. **Create a Google Form:**
   - Go to https://forms.google.com
   - Create a new form
   - Add fields: Name, Email, Message
   - Make sure the form is set to accept responses

2. **Get the Embed Link:**
   - Click the "Send" button (top right)
   - Click the "</>" (embed) icon
   - Copy the iframe code
   - Extract the `src` URL from the iframe
